The **Open Database License (ODbL)** is specifically designed for databases and their contents.

**You are free to:**
- **Share** — Copy and redistribute the database
- **Create** — Produce works from the database
- **Adapt** — Modify, transform, and build upon the database

**Under the following conditions:**
- **Attribution** — You must attribute the database and keep existing copyright notices intact
- **Share-Alike** — If you publicly distribute an adapted database, you must do so under the same ODbL license
- **Keep Open** — If you redistribute the database or an adapted version, you must keep it open and available

**Best for:** Large database collections where you want to ensure derivative databases remain open and available under the same terms

**Common use cases:**
- Geographic data collections
- Transport network databases
- Statistical datasets
